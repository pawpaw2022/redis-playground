## CLI 

`redis-cli` start CLI 
`quit` exit 

`SET key value` store key-val pair
`GET key` retrieve key-val pair 
`DEL key` delete 
`EXISTS key` check existance

`KEYS *` show all keys 
`flushall` delete all keys  

`SET key value ex 10` set expiration time for 10 sec
